# MASSIV — мебель из массива дерева

Имиджевый сайт мебельной мастерской MASSIV с кинематографическими видео-секциями, плавной прокруткой и адаптивной фотогалереей проектов.

## Технологии

- Next.js 14
- React 18
- TypeScript
- GSAP и ScrollTrigger
- Lenis

## Локальный запуск

Требуется Node.js 18.17 или новее.

```bash
npm install
npm run dev
```

После запуска откройте [http://localhost:3000](http://localhost:3000).

## Production-сборка

```bash
npm run build
npm run start
```

## Публикация на Vercel

1. Загрузите репозиторий на GitHub.
2. Импортируйте его в Vercel.
3. Vercel автоматически определит Next.js и выполнит `npm run build`.

Переменные окружения для текущей версии проекта не требуются.
